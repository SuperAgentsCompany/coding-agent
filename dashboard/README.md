# Super Coding Agent Dashboard

## Overview
This React + Vite dashboard is an internal tool to visualize, observe, and interact with the advanced code generation and DPO alignment metrics of the fine-tuned `gemma4-coding` model. It features a "Code Stream" interface with reasoning capabilities and agentic control elements (Pause, Step-through, Feedback injection).

## Technology Stack
- **Framework**: React + Vite + TypeScript
- **Styling**: Vanilla CSS (Nova Design System conventions)
- **Deployment**: Primarily localized development tool.

## Getting Started
To run the development server locally:

```bash
npm install
npm run dev
```

For more detailed architectural documentation, please refer to the [FRONTEND_ARCHITECTURE.md](https://github.com/SuperAgentsCompany/documentations/blob/main/engineering/frontend_architecture.md) in the central documentation repository.
